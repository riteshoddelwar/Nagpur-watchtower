# Nagpur City Watch 📍🚔  AI-assisted, anonymous community safety platform for Nagpur. Citizens can report local incidents (theft, suspicious activity) securely.   - **Interactive Map:** Free OpenStreetMap pins. - **Strict Moderation:** Admin queue prevents fake spam/gossip. - **100% Anonymous:** No tracking.  *Not a 100/112 emergency replacement.
